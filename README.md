# Outdoor Ready

Outdoor Ready is a focused utility app for CP3406 Assessment 1. It gives users
an at-a-glance answer to one question: how should I prepare before going
outside right now?

## Planned core features

- one-screen readiness summary driven by live weather data
- settings that directly change the summary and visible metrics
- Jetpack Compose + Material 3 UI
- ViewModel + Repository + Hilt + Retrofit architecture

## Status

This repository contains the current implementation for CP3406 Assessment 1 and
is ready for continued development, testing, and submission packaging.
